Startrail is an experimental network designed and operated by Rodrigo Laneth for personal use and research into next-generation communication protocols.

Built for tomorrow.

The Startrail network is IPv6-only and uses the address range 2a06:b700:1:100::/56, which is provided by Miralium Research. It achieves connectivity with IPv4-only internet services through 464XLAT. The NAT64 gateway, hosted on Vultr, has the public IPv4 address 216.238.109.96.
